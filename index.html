    What is AKKADIAN?

    The AKKADIAN is set out to create a world-class decentralized shared network with shared computing ecosystem using blockchain and existing idling resources such as bandwidth, storage and computing powermesh network technologies. Spiderweb solves the cost and accessibility issues so that Internet access is no longer bounded by existing service provider’s network, we will change the way people are connected. The project will begin utilizing existing, but idle resources from existing infrastructure such as fiber optic network, 4G and 5G cellular networks, and soon SpaceX’s Starlink LEO satellites communication system. Every user will be able to access our network at a much-reduced rate than any current ISP can provide. Every user will become a public node of Spiderweb network and share their resources to improve utilization for both wired and wireless Internet services from existing ISP, improve network coverage, as well as decrease broadband, computing and data storage fees.
